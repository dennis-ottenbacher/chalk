# Hardware & External Integrations

## Hardware Setup

- **POS Periphery**:
    - **Printers**: Thermal receipt printers using ESC/POS standard.
    - **Scanners**: Handheld USB/Bluetooth scanners or integrated tablet cameras for QR code processing.
    - **Cash Drawers**: Standard trigger-based drawers.
- **Access Control**: V1 relies on staff verification via the terminal. Electronic turnstiles (Gantries) are considered for future phases but not included in the initial implementation.

## External Integrations

- **Competition Platform**:
    - Synchronize user data for competition participants to streamline check-in.
    - Specialized role: **Athlet** for specific access levels and tracking.
- **Mobile Ecosystem**:
    - Apple Wallet and Google Pay support for digital membership cards/QR codes.
